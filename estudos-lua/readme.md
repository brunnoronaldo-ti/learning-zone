# **lua** repository

This repository stores code and files for the **Lua** language. The **Lua** programming language is used in Roblox game programming; Roblox is a gaming platform. The programming language used in Roblox is **Luau**, an enhanced and faster version of the **Lua** 5.1 language.

## more information

### Brazilian Origin:

the **Lua** language, which serves as the basis for **Luau**, was created in Brazil by researchers at PUC-Rio in 1993.

### how are **LUAU**:

It's a scripting language derived from **Lua**, but with performance improvements and extra features, such as an optional typing system that helps prevent errors in the code.

### Ease of use: 

It's known for being lightweight, fast, and easy to learn, making it ideal for those starting out in creating games on Roblox Studio.

### Types of Scripts:

In Roblox, you use this language in three main types of scripts:

- **Script**: Executed on the server.
- **LocalScript**: Executed on the player's computer/mobile phone (client).
- **ModuleScript**: Used to create code that can be reused by other scripts.
