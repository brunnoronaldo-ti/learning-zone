<h1 align="center">Mini-Projects</h1>

Uma coleção de projetinhos simples que marcaram o começo da minha jornada na programação.
Cada mini-projeto aqui representa um passo, uma ideia testada, um erro que ensinou alguma coisa e um código que, de alguma forma, abriu caminho para o próximo.

Este diretório existe como memória, como registro e como referência rápida.
É onde guardo exemplos básicos, experimentos e pequenos exercícios que me ajudaram a entender melhor lógica, estrutura e boas práticas.

<h2>O que você vai encontrar aqui</h2>

<ul>
  <li>Pequenos scripts e protótipos feitos durante estudos.</li>
  <li>Projetos curtos criados para treinar conceitos específicos.</li>
  <li>Experiências com linguagens, frameworks e bibliotecas.</li>
  <li>Códigos simples que servem como base para ideias futuras.</li>
</ul>

<h2>✎Por que manter esses projetos?✎</h2>

Porque evolução não é só o que a gente cria hoje — é o que a gente já criou também.
Esses mini-projetos mostram o caminho percorrido, ajudam a revisar fundamentos e servem como inspiração para algo maior.

Cada pasta corresponde a um mini-projeto independente.
